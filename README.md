# Tattoo Studio Database

Projeto de modelagem de banco de dados para um sistema de agendamento de tatuagens.

## Estrutura

O banco possui as seguintes entidades:

- Cliente
- Tatuador
- Agendamento
- Tatuagem
- Ficha de Anamnese

## Tecnologias

- MySQL
- SQL (DDL)

## Conceitos aplicados

- Modelagem relacional
- Chaves primárias
- Chaves estrangeiras
- Integridade referencial
